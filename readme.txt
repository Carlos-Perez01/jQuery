/*Glosario de terminos

    Árbol HTML --> Representación de una página HTML en forma de árbol 
    jerárquico cuyo elemento raíz es la etiqueta <HTML>
    Script --> Elmento ejecutable dentro de mi página HTML, normalmente en js
    y que se incluye dentro de la etiqueta <script> 
    Evento --> Acción que sucede en un navegador y que está relacionada con el 
    procesos del propio navegador (Carga de una página, impresión, error), o 
    con interacciones del usuario que visualiza una página web.
    Ajax --> Comunicación asíncrona (en segundo plano) de una web que me 
    van a permitir actualizar ciertas partes de mi página web sin tener que 
    recargar la página.

*/

/*¿Qué es jQuery?

    Te permite hacer con menos codigo acciones de js, pero fundamentalmente
    manejar el lenguaje DOM de nuestra página.
    Además podremos actualizar la información sin volver a recargar la página

*/

/*DOM

    Es un API para documentos HTML y XML. Proporciona una representación 
    estructucturada del documento permitiendo la modificación de su contenido 
    o presentación visual... comunica las páginas web con los scripts...

*/

/*CDN

    Un CDN es una red de servidores distribuidos geográficamente que actúan
    como "cache" para mejorar la velocidad de navegación de los usuarios.

*/

/*jQuery UI

    Librería de componentes para jQuery que permite añadir widgets, plugins
    y efectos a nuestra página web.

*/

/*Selectores

    Para poder dotar de vida e interactividad a nuestra página web lo primero
    es seleccionar el elemento/s con los que voy a trabajar (Selectores CSS3 y
    Selectores propios)

*/

/*Arbol HTML

    Padre: El nodo inmediatamente superior.
    Hijos: Todos los nodos que se encuentran debajo justo un nivel de un elemento.
    Antecesores: El padre del padre, el padre del padre del padre etc…
    Sucesores: Los hijos, los hijos de los hijos etc…
    Hermanos: Todos los nodos que tienen el mismo padre.
    
*/


/*jQuery UI 

Es una tecnología construida sobre jQuery que nos va a permitir dotar de interactividad compleja a 
mi página web de manera más fácil ya que añade numerosas características nuevas a lo que ya conocemos 
de jQuery.

*/
